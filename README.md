# Stock-Price-Prediction-
## Project Overview
This project predicts future stock prices using historical stock data.  
Models used: **Linear Regression**  

## Features Used
- Open, High, Low, Close, Volume  

## Workflow
1. Download historical stock data using `yfinance`.  
2. Preprocess data and handle missing values.  
3. Train models: Linear Regression .

5. Plot predicted vs actual prices.
